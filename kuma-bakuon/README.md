# クマ撃退ブザー（爆音）｜あきた見守り

クマに遭遇したとき、ボタンを押すと大音量の警報音（サイレン）が鳴る LINE ミニアプリ。
高齢者でも迷わず使えるよう、操作は極限までシンプルにしている。

- **メイン**: 画面いっぱいの赤いボタン。押している間（または一度タップでトグル）大音量のサイレンが鳴る。
- **熊よけ鈴モード**: トグル ON で 3 秒ごとに「チリン」と鳴り続け、歩行中にクマへ人の気配を知らせる。
- **遭遇時ガイド**: 「さわがず・走らず・ゆっくり離れる」を大きく表示。
- 音はすべて **Web Audio API で合成**しているため、音声ファイル不要・オフラインでも鳴る。

## 開発

```bash
npm install
npm run dev
```

## 環境変数

`.env.example` を `.env` にコピーして設定する。

```
VITE_LIFF_ID=   # LINE Developers で発行した LIFF ID
```

LIFF ID が未設定でもアプリ自体は動作する（LIFF 初期化をスキップ）。

## 利用イメージ

LINE のリッチメニュー「クマ撃退ブザー」から開く想定。
端末の音量を最大にしてから使用する（画面上部に案内を表示）。

## 注意

サイレンは人にも非常に大きく聞こえる。近くに人がいないか確認してから鳴らすこと。
本格的な熊対策の代替ではなく、補助的な威嚇・予防ツール。
