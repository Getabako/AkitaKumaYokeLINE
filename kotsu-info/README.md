# 交通・くらし情報｜あきた見守り

地域の **通行止め・除雪・バス運休・クマ起因の道路規制** などの暮らし安全情報を、
**一覧と地図**でまとめて見られる LINE ミニアプリです。高齢者でも見やすいよう、
文字を大きく・操作を簡単にしています。

- カテゴリ（すべて／通行止め／除雪／バス運休／クマ規制）で大きなトグルでしぼりこみ
- カードをタップすると、地図がその場所にパンしてポップアップ表示
- 「クマ規制」は、クマ出没で通行止めになっている道路を示します（クマ見守りアプリ群と連動）

## 開発

```bash
npm install
npm run dev      # Vite 開発サーバー
npm run build    # 本番ビルド
```

## 環境変数（.env）

`.env.example` をコピーして `.env` を作成してください。

| 変数 | 説明 |
|------|------|
| `VITE_LIFF_ID` | LINE LIFF の ID。未設定でもアプリは動作します（LIFF 初期化をスキップ） |
| `VITE_API_BASE` | 情報取得 API のベース URL。設定すると `${VITE_API_BASE}/api/info` を取得。未設定時は秋田県内のサンプルデータを表示 |

## 使い方（運用）

LINE リッチメニューの「交通・くらし情報」から開く想定です。

## 本番運用について

現在表示されるのはサンプル（モック）データです。
本番では **自治体・交通事業者のデータ連携**（API）を `VITE_API_BASE` 経由で接続し、
リアルタイムの通行止め・除雪・運休・クマ規制情報を配信することを想定しています。
地図は OpenStreetMap タイル（API キー不要）を使用しています。
