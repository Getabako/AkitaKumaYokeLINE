# クマに遭わない暮らし方ガイド（kuma-guide）

「あきた見守り」クマ対策アプリ群の **本体価値** を担う LINE ミニアプリ。
クマの居場所を当てるのではなく、**クマに「遭わない」暮らし方の知識**と、
**きょうのクマ危険度**を、高齢者にも分かるように大きな文字で届ける。

リッチメニュー6マス目「つかいかた」からの飛び先を想定。

## 画面

- **トップ**：大きな3つのメニュー
- **クマに遭わない6つのこと**：時間／季節／場所／天気／行動／家のまわり（理由つき）
- **もしクマに会ったら**：さわがない・ゆっくり後ずさり 等
- **きょうのクマ危険度**：現在時刻と月から「高い／ふつう／低い」を簡易判定（`src/lib/risk.ts`）

## 開発

```bash
npm install
npm run dev      # Vite dev server
npm run build    # 本番ビルド
```

## 環境変数

`.env.example` を `.env` にコピー。

```
VITE_LIFF_ID=   # LINE LIFF ID（未設定でもアプリ自体は動く）
```

## デプロイ

Vercel（`vercel.json` 準拠）。GitHub push で自動デプロイ。

## 備考

危険度は時間帯＋季節のみの簡易版。将来は天気APIや地域のドングリ豊凶を足して本実装する余地を残してある。
