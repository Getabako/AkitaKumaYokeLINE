# クマ避けAKITA デザイン共通仕様（全アプリ統一）

全6アプリで見た目を統一するための共通仕様。各アプリはこのCSS・フォント・装飾を**そのまま同じに**適用する。
狙い：野暮ったさを消し、丸ゴシックの親しみ＋立体的な装飾＋はっきりした区切りで「イケてる・分かりやすい」に。スマホ縦持ち最適化。

重要ルール：
- アプリ内の「あきた見守り」という表記は**すべて「クマ避けAKITA」に置換**（古い名前。タイトル・ヘッダー・本文すべて）。
- 絵文字は引き続き一切使わない（装飾はCSSで作る）。
- 厳格TS・テンプレ作法を壊さない。

---

## 1. フォント（各 index.html の `<head>` に追加）

```html
    <link rel="preconnect" href="https://fonts.googleapis.com" />
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
    <link
      href="https://fonts.googleapis.com/css2?family=M+PLUS+Rounded+1c:wght@500;700;800;900&family=Zen+Kaku+Gothic+New:wght@400;500;700;900&display=swap"
      rel="stylesheet"
    />
```

- 見出し：`M PLUS Rounded 1c`（丸ゴシック・親しみと迫力）
- 本文：`Zen Kaku Gothic New`（読みやすい角ゴシック）

`<title>` は「（アプリ名）｜クマ避けAKITA」に統一。

## 4. マークアップの当て方（各アプリ共通の作法）

- ルート：`<div className="screen">`
- ヘッダー：
  ```tsx
  <header className="app-header">
    <span className="brand-tag">クマ避けAKITA</span>
    <h1 className="app-title mt-2">クマ出没マップ</h1>
    <p className="app-sub">秋田の安心を、いつものLINEで</p>
  </header>
  ```
- セクション見出しは `<h2 className="section-label">…</h2>`、区切りは `<div className="divider" />`。
- カードは `panel` か `panel panel-accent`。主要ボタンは `btn-primary`、副ボタンは `btn-outline`。
- 見出しは丸ゴシック（h1/h2/h3 で自動）。強調したいタイトルに `emboss` を足す。
- 余白はゆったり（`space-y-5`〜`space-y-6`、`p-5`〜`p-6`）。スマホ縦持ち前提、`max-w-md` を維持。
