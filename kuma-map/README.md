# クマ出没マップ（あきた見守り）

秋田県のクマ・イノシシ・シカの出没情報を、地図で大きく・見やすく確認できる LINE ミニアプリ（LIFF）です。高齢者の方でも読みやすいよう、文字・ボタン・行間をすべて大きめに設計しています。

- 秋田県中心の地図（OpenStreetMap・APIキー不要）にピンを表示
- 獣種（クマ／イノシシ／シカ）と種別（目撃／痕跡／人身被害）で絞り込み
- ピンをタップすると、日時・種別・信頼度（うそ情報チェック済みの旨）を大きく表示

## 開発・起動

```bash
npm install
npm run dev      # Vite 開発サーバー
npm run build    # 本番ビルド（tsc -b && vite build）
```

## 環境変数（`.env`）

`.env.example` をコピーして `.env` を作成し、以下を設定します。

| 変数 | 説明 |
|------|------|
| `VITE_LIFF_ID` | LINE Developers で発行した LIFF ID。未設定でもアプリは動作します（LIFF 初期化をスキップ）。 |
| `VITE_API_BASE` | 出没情報 API のベース URL。設定すると `${VITE_API_BASE}/api/sightings` から GET で取得します。未設定の場合は `localStorage` を読み、空ならモックデータを seed します。 |

## 使い方（運用イメージ）

LINE のリッチメニューに「クマ地図」ボタンを置き、そこからこの LIFF アプリを開く想定です。ユーザーは地図を見るだけで、いまどこで何が出ているかを把握できます。
