# あいさつメッセージ（友だち追加時）

LINE公式アカウントに友だち追加された瞬間に自動で届くメッセージ。
高齢者にも分かるよう、やさしく・大きく・専門用語ゼロで。

## 設定する場所
LINE Official Account Manager → 対象アカウント → **ホーム → あいさつメッセージ** に貼り付け。
- `{ニックネーム}` は LINE の差し込み変数（相手の名前が自動で入る）。Managerの「ユーザーの表示名」を挿入すると入る。
- 下のメッセージ1とメッセージ2を、2つの吹き出しに分けて設定すると読みやすい。

---

## メッセージ1（あいさつ＋このアカウントの説明）

```
{ニックネーム}さん、友だち追加ありがとうございます🐻

「クマ避けAKITA」です。
秋田のクマの出没から、ご家族の毎日を守るお手伝いをします。

むずかしいことはありません。
新しいアプリを入れなくても、このLINEだけでつかえます。
```

---

## メッセージ2（つかい方の案内）

```
▼ 下のメニューから、ボタンを押すだけ

🗺 クマ地図
　近くのクマ出没を地図で見られます

📣 目撃をしらせる
　クマを見かけたら、ボタンひとつで知らせられます

🔊 クマ撃退ブザー
　大きな音で、クマを追いはらいます

🔔 通知エリア設定
　お住まいの地域でクマが出たら、すぐお知らせします

🚧 交通・くらし情報
　通行止め・バス・除雪などのお知らせ

❓ つかいかた
　クマに遭わないコツと、きょうの危険度

ご家族にも、このアカウントをすすめてあげてください。
```

---

## 短い1通だけにしたい場合（メッセージ1本版）

```
{ニックネーム}さん、友だち追加ありがとうございます🐻

「クマ避けAKITA」です。
クマの出没から、ご家族の毎日を守ります。
新しいアプリは要りません。このLINEだけ。

下のメニューのボタンを押すだけでつかえます。
・クマ地図　・目撃をしらせる　・クマ撃退ブザー
・通知エリア設定　・交通くらし情報　・つかいかた

ご家族にもすすめてあげてください。
```

---

## メモ
- 絵文字は機種により表示が変わるので、入れすぎない。上記くらいが安全。
- 「あいさつメッセージ」とは別に、**リッチメニュー**（下の6マス）も忘れず「表示する」にしておく（`セットアップと運用ガイド.md` 手順D）。
- 将来 line-harness を繋いだら、あいさつのあとに「お住まいの地域を登録しましょう」と通知エリア設定へ誘導するステップ配信を足すと、登録率が上がる。
